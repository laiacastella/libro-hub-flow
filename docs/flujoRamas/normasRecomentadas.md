# 📌 Normas recomendadas

## En `main`

* No hacer push directo.
* Solo entrar mediante Pull Request desde `staging`.

## En `staging`

* Solo recibir cambios desde `develop`.

## En `develop`

* No subir cambios sin revisar localmente.

## En ramas feature

* Una rama = una tarea.
* Borrar la rama al terminar.
